It's an emulator, innit?

About

GBCC is a cross-platform Game Boy and Game Boy Color emulator written in C, with a focus on accuracy. Available on Linux and Android.

Features

Core

  • Memory Bank Controllers:
    • MBC1
    • MBC2
    • MBC3
    • MBC5
    • MBC6
    • MBC7
    • HuC1 (IR port not implemented)
    • HuC3 (IR port not implemented)
    • MMM01
    • Sachen MMC1
    • Sachen MMC2
